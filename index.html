Що таке GNU/Linux?

GNU/Linux — це відкрита операційна система, що поєднує ядро Linux і програмне забезпечення з проекту GNU. Вона є однією з найбільш популярних систем серед розробників, серверів і науковців завдяки своїй гнучкості, стабільності і безпеці.

ОС GNU/Linux підтримує величезну кількість програм і технологій, від базових утиліт до складних інструментів для розробки, адміністрування серверів, роботи з науковими даними, тощо. Вона також є основою для багатьох інших операційних систем, таких як Android.

Популярні дистрибутиви

Існує безліч дистрибутивів GNU/Linux, кожен з яких має свої особливості і цільову аудиторію. Ось деякі з найбільш популярних:

  • Ubuntu — один із найпопулярніших дистрибутивів, що фокусується на простоті використання.
  • Debian — відомий своєю стабільністю та безпекою, часто використовується для серверів.
  • Fedora — дистрибутив, який завжди включає найновіші версії програмного забезпечення.
  • Arch Linux — для досвідчених користувачів, забезпечує максимальну гнучкість та контроль.
  • Linux Mint — зручний для початківців, побудований на базі Ubuntu.
  • CentOS — корпоративний дистрибутив на базі Red Hat Enterprise Linux, популярний серед серверів.

Основні команди в CLI Linux

Робота в командному рядку (CLI) є одним з основних способів взаємодії з системою Linux. Ось кілька основних команд: 

            
# Перегляд поточної директорії
pwd

# Виведення тексту на екран
echo "Hello, Linux!"

# Перегляд вмісту поточної директорії
ls

# Перехід до іншої директорії
cd /home/user

# Створення нового файлу
touch newfile.txt

# Створення нової директорії
mkdir new_directory

# Копіювання файлів
cp file1.txt file2.txt

# Переміщення файлів
mv oldfile.txt newfile.txt

# Видалення файлу
rm file_to_delete.txt

# Перегляд історії виконаних команд
history
