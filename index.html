Що таке GNU/Linux?

GNU/Linux — це потужна та гнучка операційна система з відкритим вихідним кодом. Вона складається з ядра Linux, розробленого Лінусом Торвальдсом, та програмного забезпечення з проекту GNU, яке надає інтерфейс користувача, утиліти, бібліотеки та інше.

Приклади команд у CLI

            
# Перегляд поточної директорії
pwd

# Виведення повідомлення
echo "Hello, Linux!"

# Перегляд вмісту директорії
ls

# Перехід в іншу директорію
cd /home/user

# Створення нового файлу
touch newfile.txt

# Створення нової директорії
mkdir new_directory

# Копіювання файлів
cp source.txt destination.txt

# Переміщення файлів
mv old_name.txt new_name.txt

# Видалення файлу
rm file_to_delete.txt

# Перегляд історії команд
history

Bash-скрипт для імітації авторизації

            
#!/bin/bash

echo "Введіть ім'я користувача:"
read username

echo "Введіть пароль:"
read -s password

if [ "$username" == "admin" ] && [ "$password" == "password123" ]; then
    echo "Авторизація успішна!"
else
    echo "Невірне ім'я користувача або пароль."
fi
